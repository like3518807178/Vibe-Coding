# TASK.md

## 当前版本
V1：select 多客户端聊天室

## 当前目标
实现一个基于 select 的多人聊天室服务端：

- 支持多个客户端同时连接
- 任意客户端发送文本，其他客户端可以收到广播
- 任意客户端断开，服务端正确清理连接并继续服务其他客户端

## 必做功能
1. 使用 select() 做 I/O 多路复用
2. 维护 fd_set 和 maxfd
3. 维护客户端集合
4. 广播消息给其他在线客户端
5. 客户端断开时清理 fd

## 当前限制
1. 只允许完成 V1
2. 不要实现 V2 及之后内容
3. 不要加入 JSON
4. 不要加入长度前缀 framing
5. 不要加入 SQLite
6. 不要加入 epoll
7. 不要加入线程池

## 验收标准
1. 启动服务端
2. 开 3 个终端分别 nc 连接
3. A 发消息，B/C 能收到
4. 任意一个客户端断开，其他客户端仍能继续通信

## 完成后必须输出
1. 修改文件列表
2. 关键实现说明
3. 编译命令
4. 运行命令
5. 验收步骤
6. 当前版本故意没做什么
