# TinyIM V1

当前版本只实现 V1：基于 `select` 的多客户端聊天室服务端。

## 功能范围

- 监听 `7777` 端口
- 支持多个客户端同时连接
- 任意客户端发送文本，其他客户端可收到广播
- 客户端断开后，服务端正确清理连接并继续服务

## 不在本版本内的内容

- 长度前缀 framing
- JSON
- 数据库
- 登录注册
- `epoll`
- 线程池

## 编译

```bash
cmake -S . -B build
cmake --build build
```

## 运行

```bash
./build/tinyim_v1
```

## 验收

1. 启动服务端。
2. 新开 3 个终端，分别执行 `nc 127.0.0.1 7777`。
3. 在 A 终端输入任意文本，确认 B/C 可以收到广播。
4. 任意一个客户端按 `Ctrl-D` 断开。
5. 确认其他客户端仍可继续通信。
