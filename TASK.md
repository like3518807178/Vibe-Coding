# TASK.md

## 当前版本
V0：单连接 Echo 原型

## 当前目标
实现一个单进程 TCP Echo Server：

- 接受 1 个 TCP 客户端连接
- 客户端发什么，服务端回什么
- 客户端断开后服务端不崩溃
- 服务端可以继续等待下一个连接

## 必做功能
1. socket()
2. bind()
3. listen()
4. accept()
5. read()
6. write()/writeAll()

## 必须正确处理的情况
1. read() == 0：对端关闭连接
2. read() < 0：读取出错
3. 服务端打印清晰错误信息
4. 当前客户端断开后，服务端继续 accept 下一个客户端

## 文件要求
建议新增或修改以下文件：

- CMakeLists.txt
- src/main.cpp
- src/server.cpp
- src/server.h
- README.md

如果仓库里还没有这些文件，可以直接补齐最小可运行版本。

## 当前限制
1. 只允许完成 V0
2. 不要实现 V1 及之后的内容
3. 不要加入 select
4. 不要加入 epoll
5. 不要加入 JSON
6. 不要加入 SQLite
7. 不要加入线程池
8. 不要为了后续版本提前设计复杂架构

## 验收标准
1. 服务端可监听 7777 端口
2. 使用 nc 连接：nc 127.0.0.1 7777
3. 输入任意文本，服务端原样返回
4. 客户端 Ctrl-D 断开后，服务端继续运行
5. 新客户端仍可再次连接并回显

## 完成后必须输出
1. 修改文件列表
2. 关键实现说明
3. 编译命令
4. 运行命令
5. 验收步骤
6. 当前版本的边界（哪些功能故意没做）
EOF
