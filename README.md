# TinyIM V0

当前版本只实现 V0：单连接 TCP Echo Server。

## 功能范围

- 监听 `7777` 端口
- 接受 1 个客户端连接
- 客户端发送什么，服务端原样回显什么
- 客户端断开后，服务端继续等待下一个连接

## 不在本版本内的内容

- 多客户端并发
- `select` / `epoll`
- 协议封包
- JSON
- 数据库
- 登录注册
- 线程池

## 编译

```bash
cmake -S . -B build
cmake --build build
```

## 运行

```bash
./build/tinyim_v0
```

## 验收

1. 启动服务端。
2. 新开终端执行 `nc 127.0.0.1 7777`。
3. 输入任意文本，确认服务端原样返回。
4. 按 `Ctrl-D` 断开客户端。
5. 再次执行 `nc 127.0.0.1 7777`，确认仍能连接并继续回显。
